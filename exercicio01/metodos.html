Métodos de Requisição HTTP

Método GET

       Esse é o método mais comum entre todos, pois por ele que solicitamos a representação de uma página web.

Método POST

       Esse método é utilizado quando queremos criar um recurso, com isso, quando-o utilizamos os dados vão no corpo da requisição e não na URI.

Método PUT

       Requisita que um recurso seja "guardado" na URI fornecida, caso o recurso já existir, ele deve ser atualizado e se não existir, ele deve ser criado.

Método DELETE

        O método DELETE remove um recurso específico.
